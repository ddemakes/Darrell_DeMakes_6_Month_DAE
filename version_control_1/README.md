# Version Control 1

This directory contains exercises and examples related to **Version Control Systems**, primarily focusing on Git. It's designed to help understand and practice fundamental version control operations.

## Contents:
- **`roll_die.py`**: A simple Python script, likely used as a practical file to apply version control concepts such as:
    - Initializing a Git repository.
    - Staging and committing changes.
    - Creating and switching branches.
    - Merging branches.
    - Viewing commit history.

This folder is essential for developing proficiency in managing code changes, collaborating with others, and maintaining project history using version control.
