# Python 1

This directory contains fundamental Python exercises and scripts. It serves as a collection of basic Python concepts and programming constructs, distinct from larger project-specific Python files.

## Contents:
- **`for_loop.py`**: An example demonstrating the usage of `for` loops in Python.
- **`roll_die.py`**: A simple Python script for simulating a dice roll, often used for introductory programming exercises.

This folder is ideal for reviewing basic Python syntax and understanding how core programming principles are implemented in Python.
